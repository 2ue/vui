# 目的

# css变量命名规则
## 通用规则

- 所有命名都应添加前缀`vui`，以避免命名冲突
- 以中划线（-）链接

## 汉英对照

- 弹窗-pop up
- 遮罩-shade
- 模态-modal
- 模态框-modal dialog

## box 层级

- warp
- box
- inner(body)
- header/content(body)/footer

## layout 关系

> 与box结合
